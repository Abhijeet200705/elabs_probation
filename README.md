## 📌 Task Objective
Perform data analysis using Microsoft Excel and create:

- Pivot 1 – Summary by Category & Region

---

## 📊 Excel Analysis

### 🔹 Pivot 1: Category & Region Summary
- Rows: Category  
- Columns: Region  
- Values: Sales / Profit (Summarized)  
- Chart: Column Chart  

**Purpose:**  
To analyze sales performance across different product categories and regions.

---

### 🔹 Pivot 2: Monthly Sales Trend
- Rows: Order Date (Grouped by Month)  
- Values: Sales  
- Chart: Line Chart  

**Purpose:**  
To identify sales trends and patterns over time.
